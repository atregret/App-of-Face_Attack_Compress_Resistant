# vue

## Project setup
```
npm install
```
### 为Vue项目添加Bootstrap
npm i bootstrap@3 -S
npm i jquery -S
npm i popper.js -S

### 运行Vue项目
```
npm run serve
```

### 打包Vue项目
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
